# SmartOS AI - Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Navigate to AI Predictions
Click **"AI Predictions"** in the sidebar (✨ sparkle icon)

### Step 2: Generate Forecast
Click the **"Generate AI Forecast"** button

### Step 3: Review Results
Read the AI analysis and follow recommendations

---

## 📊 What You'll See

### Top Priority Items:
1. **Risk Level** - Check this first (LOW/MEDIUM/HIGH/CRITICAL)
2. **System Summary** - Overall status
3. **Anomalies** - Unusual patterns requiring attention
4. **Recommendations** - Specific actions to take

### Detailed Analysis:
- Smart Lighting Insights
- Security Analysis
- Irrigation Analysis
- Traffic Analysis
- Device Health

---

## ⚡ Quick Actions

### If Risk Level is CRITICAL:
1. Read System Summary immediately
2. Check Anomalies section
3. Implement recommendations NOW
4. Monitor system closely

### If Risk Level is HIGH:
1. Review Anomalies
2. Plan to implement recommendations within 1-2 hours
3. Check specific sensor pages for details

### If Risk Level is MEDIUM:
1. Note recommendations
2. Implement during next maintenance window
3. Monitor trends

### If Risk Level is LOW:
1. Review for optimization opportunities
2. Check Energy Savings estimate
3. Consider preventive actions

---

## 💡 Pro Tips

✅ **Generate forecasts every 1-2 hours** during active operations  
✅ **Act on recommendations in order** (they're prioritized)  
✅ **Check Energy Savings** for cost reduction opportunities  
✅ **Monitor Confidence score** - higher = more reliable  
✅ **Cross-reference with sensor pages** for detailed data  

---

## 🔧 Troubleshooting

**No analysis showing?**  
→ Click "Generate AI Forecast" button

**"Error: AI connection timeout"?**  
→ Check internet connection and `.env` file

**Strange predictions?**  
→ Wait for more sensor data to accumulate
